# Farm On Demand

A React + Vite web app for showcasing an on-demand farming service. This repository contains the full project, including the Vite app in the `farm/` folder.

## Features
- Responsive landing page and auth screens
- Modular React components
- Vite build setup with ESLint

## Tech Stack
- React
- Vite
- CSS

## Getting Started

Prerequisites:
- Node.js 18+ recommended
- npm

Install and run:

```bash
cd farm
npm install
npm run dev
```

Build for production:

```bash
cd farm
npm run build
npm run preview
```

## Project Structure

```
react_project/
  farm/           # Vite app
    src/
    public/
    package.json
```

## Scripts (from farm/)
- `npm run dev` - Start dev server
- `npm run build` - Production build
- `npm run preview` - Preview build
- `npm run lint` - Lint
